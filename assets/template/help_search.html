검색어를 이용해서 원하는 결과를 검색할 수 있습니다.

검색 범위를 Task로 제한하고 싶다면 Task를 All에서 원하는 Task로 선택해주세요.

검색어1   검색어2   ... : 검색어를 연속을 입력하면, 매 검색어가 포함된 결과가 검색됩니다.

user:김한웅 : 아티스트명을 검색어로 사용할 수 있습니다.

user:notassign : 아티스트가 설정되지 않은 리스트를 검색합니다.

All all 올 미ㅣ 전체 : 전체 데이터를 검색합니다.

shot 샷 전샷 전체샷 : 샷만 검색합니다.

asset assets 에셋 : 에셋리스트를 보여줍니다.

0109 : 당일 마감되어야 하는 테스크 마감일, 샷 마감일을 검색합니다.

2020-01-09 : 해당 날짜의 데일리 mov를 검색합니다.

1권 ... 8권 으로 검색이 가능합니다.

name:SS_0010 : 명확하게 샷, 에셋 네임을 검색하고 싶을 때 사용합니다.

rnum:A0001 : 1권 첫번째 샷을 의미하는 rnum:A0001 형태로 검색하면 해당 Roll Number를 검색할 수 있습니다.

tag:태그명 : 태그명으로 태그검색이 가능합니다.

deadline2d:2020-01-30 : 마감일2D 2020-01-30 샷 검색

deadline3d:2020-01-30 : 마감일3D 2020-01-30 샷 검색

2D 3D : 각각 2D, 3D샷이 검색됩니다.

00:12:23:12 : 타임코드를 검색합니다.(스캔 타임코드, 작업 타임코드)

tasks.light.status:done : light 테스크의 상태가 done인 값만 검색

task:concept task:light 김한웅 : concept과 light에 김한웅이 할당된 항목 검색

user:김한웅 or user:박지섭 : 이름이 김한웅,박지섭 둘다 들어간 Task 검색

season:S01 : 시즌명이 S01인 아이템을 검색

episode:E01 : 에피소드명이 E01인 아이템을 검색
